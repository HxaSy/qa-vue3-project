### 一、项目描述
一个简单的答题 Web ，没有华丽的 UI 与丰富的功能，有的只是答题。

### 二、已有功能
1. 题目作答
2. 查看结果
3. 待定

### 三、相关技术
前端: 
1. Vue 3 + TypeScript
2. Vite
3. Vue-Router

组件库: NutUI

后端: 待做

### 四、项目启动
```javascript
// 项目调试
// 打开终端上的网址http://127.0.0.1:5173/即可
npm run dev

// 项目打包
npm run built
```
